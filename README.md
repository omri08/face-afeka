## Face - Afeka

Application to share posts between friedns (simple facebook clone).

(project created for learning)

## Built With

**FrontEnd**:

- Hooks
- Redux
- Sass & Ant Design
- axios

**BackEnd** :

- Node.js (Express.js)
- MongoDB
- JWT
